# Containment

A static browser game about slowing a smallpox outbreak across the Northern and Middle Colonies during the 1775-1782 North American smallpox epidemic.

## Play locally

Open `index.html` in a web browser.

## Upload

Upload these files together to any static host:

- `index.html`
- `styles.css`
- `game.js`

No build step or server framework is required.

## Design note

The board is a stylized educational map of the Northern and Middle Colonies. It is designed for game clarity rather than geographic precision.
